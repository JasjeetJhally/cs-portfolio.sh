HTML, which stands for HyperText Markup Language, is a programming language that is used to create the structure and layout of a website. In 1990, Tim-Berners-Lee had created HTML. HTML is written using tags enclosed in angle brackets.

CSS, which stands for Cascading Style Sheets, is a programming language that is used as a way to design and present a website. It was initially invented in 1996, by Hakon Wium Lie, who worked with Tim-Berners-Lee, creator of HTML. CSS is written by using a number of English keywords to specify the style being implemented.

Tags are keywords within a web page that make the browser determine how to display and format the content. Tags are written with two parts, an opening and closing part. These tags are wrapped with angle brackets, such as <>.

Examples of Tags:

The header tag is used to emphasis words in a website's text. It can be used for titles, and other headings.

The paragraph tag is used for defining a paragraph in a text.

The break tag is used for line breaks in a text.

The emphasized text tag is used for italicizing text.
