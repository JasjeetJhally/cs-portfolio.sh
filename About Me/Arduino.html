Arduino Push Button

Description of Project Model

Arduino is an electronic platform that allows for the building of interactive electronic creations. In the Arduino, is a microcontroller board which reads inputs from various pieces available in the Arduino kit. Part of the Arduino kit, is a specific programming that the Arduino reads and applies, By using the Arduino Programming Language, one can give a set of directions to the microcontroller board, which can then be used to put out a variety of outputs. For my Arduino project, I decided to create a button that, when pushed, can light up an LED light. In my case, the action of pushing the buttons is the input, while the lighting of the LED light is the output.

Materials Used

For this project, the materials that I used included two buttons, and LED light, three resistors, and seven wires. Each component utilized in this project had a separate function. The buttons served as a platform for the user to give an input. Pressing these buttons will then activates a certain amount of voltage which the Arduino will then recognize. To control the current in the Arduino, the resistors are placed. When the circuit runs in the appropriate manner, it will then light up the LED bulb.

Model

To accurately model the way that my Arduino project works, I wanted to digitally recreate the Arduino and where the different parts were placed. Next, to make the web page interactive for the user viewing the page, I wanted to add a button to my digital version of my Arduino that would model the function of the actual button. Pressing this button would make the picture of the unlit button (also pictured in the Arduino), turn into a picture of a lit bulb, similar to the process of pushing the button and having the LED lit up. Another aspect of the Arduino that I wanted to model was the flow of electrons that flow in the wires when the current is active. This was represented by the moving blue rectangular box that followed a part of the wires. To create this moving electron, I used an animation in CSS, which took into account the position of the box during a set interval of time. However, this model is not entirely accurate. While it may be true that electrons do flow into the light bulb, they are suppose to begin moving when the button is pushed, not all the time.
