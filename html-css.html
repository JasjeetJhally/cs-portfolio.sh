Hover Here to Learn About HTML

HTML is a programming language that is used to create the structure and layout of a website. HTML is an acronym for HyperText Markup Language. The HT stands for HyperText, which is a text that contains links into other texts, creating an interconnected system of links that share information. The ML in HTML stands for Markup Language. A Markup Language is a system of annotating a document so that is distinguishable from the text. Marking these pieces enables for the adjustment of pieces of code, whether that be giving emphasis to a phrase, or setting aside pieces of the text. HTML is the foundation of all web pages, and is vital for the function of these pages. The concept of HTML was thought up in 1989, by Tim Berners-Lee as he was working in the European Laboratory for Particle Physics (CERN) in Geneva, Switzerland. At first it was invented for researchers living in remote places in the world to gather information; instead, it was the publishing language for the World Wide Web. HTML is written in the form of using HTML elements consisting of tags. Generally, there is a tag before and after a piece of content is written, such as in the tag. Before the content is written, a is placed before the code, and a is placed after the code. Notice how a slash is placed in the end tag, this is to distinguish that this an end tag. In some cases, there is no end tag needed, and only a beginning tag is used in this case.

Hover Here to Learn About CSS

As HTML grew, the idea of presenting the webpages was brought up. Using HTML provided for information to be shared, and the design that could be done was never drastic. To solve this problem, a new language was created to focus solely on the design factor on web pages, this was CSS. CSS, which stands for Cascading Style Sheets, is a programming language that is used as a way to present a website in a more attractive way. It was initially invented in 1996, by Hakon Wium Lie, who worked with Tim-Berners-Lee, creator of HTML. Both worked at CERN, a European research facility. With CSS, web pages can be designed in a infinite amount of ways, creating unique, better looking and structured websites. CSS is written by using a number of English keywords and a set of rules to specify the style being implemented. This is done through the usage of declaration blocks (curly brackets) and selectors, which are used to declare which specific piece of text is being designed. These selectors can apply to an element found in the text, such as a header, or a specified attribute found in an element. Once these selectors are set, and the declaration blocks are placed, properties are used to design the web page, creating a more visually appealing web page.

What are tags?

Tags are keywords within a web page that make the browser determine how to display and format the content. Tags are written with two parts, an opening and closing tag. These tags are wrapped with angle brackets, such as <>.

What are attributes?

Attributes are added to tags to modify the tag to perform certain functions. They are added to the starting HTML tag. These attributes can be changed by adding certain keywords to the tag, such as class or id. For example, an id specifies an element in HTML, and can be used to customize certain aspects of that element.

Examples of Tags:

Header Tag


The header tag is used to emphasis words in a website's text. It can be used for titles, and other headings.


Paragraph Tag


The paragraph tag is used for defining a paragraph in a text. This involves seperating text by adding space in between.


Linebreak Tag


The break tag is used for line breaks in a text. This will seperate lines from each other. There is no end tag for this tag.


Italic Tag


The emphasized text tag is used for italicizing text, which can be used to differentiate text from each other.
